# 首頁：知識拼圖格

> 原名「知識地圖」。**2026-07-20** 改為拼圖取向：去掉東南西北。詳見 `knowledge-puzzle.md`。

## 摘要

每個 seed 是一塊「拼圖片」，擺在格子上組成**知識拼圖**。位置仍有相對意義，但不綁真實方位。模板可換。

## 產品規則

1. Seed = 一格上的拼圖片  
2. 空格可留白；可拖去空格或與他格交換  
3. 點一格 → 讀／改／存版／看差異  
4. **存拼圖位置** → 寫回 `docs/seeds.json`  
5. **模板可選**（空白 N×N、示範模板等）  
6. 拼圖分 **個人版**／**社群版**（見 `knowledge-puzzle.md`）

## 寫回倉庫（已做）

瀏覽器拖曳 → 設定鑰匙 → 存拼圖位置（GitHub API）。
