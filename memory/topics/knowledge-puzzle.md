# 知識拼圖（產品方向）

## 摘要

首頁不是「地理地圖」，而是**知識拼圖**：種子像拼圖片，擺在格子上組成畫面。東南西北拿掉。模板可自由選。

## 與舊說法的關係

| 舊 | 新 |
|---|---|
| 知識地圖 | **知識拼圖** |
| 東南西北方位 | **不使用** |
| 固定一種格子 | **可選拼圖模板** |

台灣橫貫公路示範仍可當「一種模板」，不再暗示真實方位。

## 兩種拼圖

### 1. 個人版

- 自己的拼圖工作台
- 可設定 **公開** 或 **私人**
- 位置自己拖、自己存
- 對應現況 Pages／seeds.json 流程

### 2. 社群版（像菜市場攤位）

- 大家搶同一張拼圖上的格子
- **位置用點數去佔**（熱門位置較貴或需較多點）
- 點數來源（之後實作）：
  - 完成任務（煉筆記、存版、分享、協助改稿…）
  - 隨時間緩慢累積
- 搶到的位置可展示自己的 seed；被更強點數擠走的規則之後再定

## 拼圖模板

使用者可自由選模板（第一版已有選單入口）：

| 模板 | 說明 |
|---|---|
| 空白 8×8／10×10／12×12 | 純格子，自己擺 |
| 示範：台灣橫貫公路 | 現有示範種子擺法 |

之後可加：時間軸、主題島、樹狀、圓環等模板，不必綁地理。

## 資料欄位（seeds.json 的 map）

```json
{
  "title": "知識拼圖",
  "kind": "personal",
  "visibility": "public",
  "template": "grid-10",
  "cols": 10,
  "rows": 10
}
```

- `kind`: `personal` | `community`
- `visibility`: 僅個人版有意義（`public` | `private`）
- `template`: 模板 id

## 導覽體感（2026-07-21 07:38 更新）

| 元素 | 意義 |
|---|---|
| **SEED** | 點開才出下拉（路徑＋最近五處） |
| **階層列** | 固定兩層：`首頁 › SEED 名稱`；點哪層就進哪層 |
| **右上模式** | `編輯`／`A4檢視`／`記錄比對`；不放進階層 |
| **狀態** | 右上狀態／可見拿掉 |

### 編輯框

- 框**內**＝可列印正文；框**下方**小字深灰＝註解（不列印）
- 框內直接改、**自動儲存**（有鑰匙寫倉庫；否則本機暫存）
- 可拖拉換序；框與框之間只有小綠色 `＋`
- 內文與註解都空白時自動刪除；每框也有刪除 icon

### 記錄比對

- 使用 A4 紙面
- 紅框＝刪除內容；綠框＝變更／新增內容

### 首頁排版

- 拖動後**自動儲存**；不再跳出 reset／save 圖示
- 拼圖下方新增「我的 SEED」：可收起、放回拼圖、新增、刪除
- 初次開啟只在拼圖放前 8 顆，其餘收在「我的 SEED」

### 已拿掉（之後再加）

- 角色選擇與名稱
- 狀態／可見、拼圖模板下載、打包帶走、還原回來、AI 鑰匙（選單入口）

## 最小互動流程（System → User → AI）

這版 AI 不做複雜系統面板：

1. **System** 先給一段情境、問題或提示
2. **User** 輸入回答
3. **AI** 根據 user 的回答直接回應

## 操作速記

- **SEED**：點開路徑選單；旁邊只顯示 `首頁 › SEED 名稱`
- **點種子**：直接編輯框；註解在框下方
- **模式**：編輯／A4檢視／記錄比對
- **我的 SEED**：收納未上拼圖的 SEED，可新增／刪除
- **拼圖與筆記**：自動儲存（需鑰匙才寫進倉庫）
- **AI 問答**：System → User → AI（對話框仍在）
