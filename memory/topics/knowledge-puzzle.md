# SEED 棋盤（產品方向）

## 摘要

首頁不是「地理地圖」，而是**知識拼圖**：種子像拼圖片，擺在格子上組成畫面。東南西北拿掉。模板可自由選。

> 2026-07-21 起，對外用語由「拼圖」改為 **棋盤**：我的 SEED 是可派上棋盤的棋子，也可拖回收納區。

## 與舊說法的關係

| 舊 | 新 |
|---|---|
| 知識地圖 | **知識拼圖** |
| 東南西北方位 | **不使用** |
| 固定一種格子 | **可選拼圖模板** |

台灣橫貫公路示範仍可當「一種模板」，不再暗示真實方位。

## 兩種拼圖

### 1. 個人版

- 自己的拼圖工作台
- 可設定 **公開** 或 **私人**
- 位置自己拖、自己存
- 對應現況 Pages／seeds.json 流程

### 2. 社群版（像菜市場攤位）

- 大家搶同一張拼圖上的格子
- **位置用點數去佔**（熱門位置較貴或需較多點）
- 點數來源（之後實作）：
  - 完成任務（煉筆記、存版、分享、協助改稿…）
  - 隨時間緩慢累積
- 搶到的位置可展示自己的 seed；被更強點數擠走的規則之後再定

## 拼圖模板

使用者可自由選模板（第一版已有選單入口）：

| 模板 | 說明 |
|---|---|
| 空白 8×8／10×10／12×12 | 純格子，自己擺 |
| 示範：台灣橫貫公路 | 現有示範種子擺法 |

之後可加：時間軸、主題島、樹狀、圓環等模板，不必綁地理。

## 資料欄位（seeds.json 的 map）

```json
{
  "title": "知識拼圖",
  "kind": "personal",
  "visibility": "public",
  "template": "grid-10",
  "cols": 10,
  "rows": 10
}
```

- `kind`: `personal` | `community`
- `visibility`: 僅個人版有意義（`public` | `private`）
- `template`: 模板 id

## 導覽體感（2026-07-21 08:25 更新）

| 元素 | 意義 |
|---|---|
| **SEED** | 點開才出下拉（路徑＋最近五處） |
| **階層列** | SEED 內顯示 `首頁 › SEED 名稱`；回首頁只顯示 `首頁` |
| **右上模式** | `編輯`／`文件檢視`／`歷程記錄`；不放進階層 |
| **狀態** | 右上狀態／可見拿掉 |

### 編輯框

- 框**內**＝可列印正文；左側同款框＝該段註解（不列印）
- 文件模板標題／副標題可改，下方只用灰字標示 `版型：文字模板`
- 編輯時只**暫存草稿**；按 `Save` 才建立正式版本
- `Save` 在編輯畫面的第二行一般工具列，不固定於頂端
- 段落編號只顯示純文字；不顯示拖曳圖示
- 正文盡量連續；刪除 icon 位於正文框右上角，新增 `＋` 位於其下方

### 記錄比對

- 只選一個編輯日期，系統自動跟上一次版本比對
- 不顯示額外頁面大標題；下拉選單與 A4 紙面同寬
- 編輯者格式為 `你－編輯－日期時間`；AI 版本才標示 `AI 編輯`
- 紅框＝刪除內容；綠框＝變更／新增內容

### 三種 SEED

- **文件模板**：人從零撰寫；以「新增文字」方式組成 A4
- **簽核模板**：分兩層——**範本**負責欄位／流程／權限／通知設定；**申請單**從範本複製後以問卷式填寫，詳見 [`approval-workflow.md`](approval-workflow.md)
- **主題討論模板**：製作 16:9 直播風格截圖；標題、副標題與主要文字都在畫面內，左下顯示最新四則評論

### A4 匯出

- 文件檢視第二行提供 `匯入檔案`、`PDF 匯出`、`Word 匯出`、`圖片匯出`
- Save 使用鮮明綠色；匯入與匯出使用另一組柔和色彩，工具列隨內容捲動、不固定頂端
- PDF 使用瀏覽器列印；Word 下載 `.doc`；圖片下載 `.png`

### 首頁排版

- 拖動後**自動儲存**；不再跳出 reset／save 圖示
- 棋盤下方是「我的 SEED」：保留最低操作高度，項目太多時區域內可上下捲動
- 「我的 SEED」與棋盤支援雙向拖放；棋盤上的 SEED 像出戰棋子
- 初次開啟只在棋盤放前 8 顆，其餘收在「我的 SEED」
- 「上下為主」與「全部棋盤」隱藏垂直捲軸
- **System Seed**（2026-07-21）：首頁獨立入口；流程為 `類型 → 範本 → 版本 → 用這版建立 SEED → 修改 → Save`
- 「＋ 新增 SEED」與舊對話框皆導向 System Seed；可從目前 SEED 或歷程版本「另存為 System Seed 範本」

### 已拿掉（之後再加）

- 狀態／可見、拼圖模板下載、打包帶走、還原回來、AI 鑰匙（選單入口）

## 最小互動流程（System → User → AI）

這版 AI 不做複雜系統面板：

1. **System** 先給一段情境、問題或提示
2. **User** 輸入回答
3. **AI** 根據 user 的回答直接回應

## 操作速記

- **SEED**：點開路徑選單；SEED 內顯示 `首頁 › SEED 名稱`，首頁只顯示 `首頁`
- **點種子**：直接編輯框；註解在框下方
- **模式**：編輯／文件檢視／歷程記錄；Save 只在編輯內容最下方
- **我的 SEED**：收納未上棋盤的 SEED，可新增／刪除／拖曳出戰；太長時區域內捲動
- **棋盤**：位置自動儲存；內容只暫存草稿，按 Save 才建立版本
- **AI 問答**：System → User → AI（對話框仍在）

## 連續正文＋文件匯入（2026-07-21 09:25 更新）

- `Save` 只在編輯模式出現，位於模式列下方的小型工具列。
- 標題與副標題可直接編輯；按 Save 時，倉庫型 SEED 也會更新 `seeds.json`。
- 文件模板維持左右關係，但不做成工具格：
  - 左側：與正文同款的註解框，編號使用純文字。
  - 右側：連續正文；刪除在框右上角，新增 `＋` 緊接在下方。
- 首頁提供三個直接切換按鈕：左右為主、上下為主、全部棋盤；預設全部棋盤。
- 棋盤 SEED 不再有向下圖示；「我的 SEED」不再有派遣按鈕，雙向移動只用拖曳。
- `A4 檢視` 改名 `文件檢視`，提供 Word、PDF、TXT、Markdown、CSV、TSV、JSON 匯入。
- 匯入時抽取文字成 SEED，原始 Word／PDF 存於瀏覽器 IndexedDB：
  - 未修改可匯出完全相同的原檔。
  - 修改後會依目前內容重新產生，不能保證原 Word 排版或 PDF 像素完全一致。
- `記錄比對` 改名 `歷程記錄`，移除比對按鈕；預設自動比較目前畫面與上次 Save。
