# 知識拼圖（產品方向）

## 摘要

首頁不是「地理地圖」，而是**知識拼圖**：種子像拼圖片，擺在格子上組成畫面。東南西北拿掉。模板可自由選。

## 與舊說法的關係

| 舊 | 新 |
|---|---|
| 知識地圖 | **知識拼圖** |
| 東南西北方位 | **不使用** |
| 固定一種格子 | **可選拼圖模板** |

台灣橫貫公路示範仍可當「一種模板」，不再暗示真實方位。

## 兩種拼圖

### 1. 個人版

- 自己的拼圖工作台
- 可設定 **公開** 或 **私人**
- 位置自己拖、自己存
- 對應現況 Pages／seeds.json 流程

### 2. 社群版（像菜市場攤位）

- 大家搶同一張拼圖上的格子
- **位置用點數去佔**（熱門位置較貴或需較多點）
- 點數來源（之後實作）：
  - 完成任務（煉筆記、存版、分享、協助改稿…）
  - 隨時間緩慢累積
- 搶到的位置可展示自己的 seed；被更強點數擠走的規則之後再定

## 拼圖模板

使用者可自由選模板（第一版已有選單入口）：

| 模板 | 說明 |
|---|---|
| 空白 8×8／10×10／12×12 | 純格子，自己擺 |
| 示範：台灣橫貫公路 | 現有示範種子擺法 |

之後可加：時間軸、主題島、樹狀、圓環等模板，不必綁地理。

## 資料欄位（seeds.json 的 map）

```json
{
  "title": "知識拼圖",
  "kind": "personal",
  "visibility": "public",
  "template": "grid-10",
  "cols": 10,
  "rows": 10
}
```

- `kind`: `personal` | `community`
- `visibility`: 僅個人版有意義（`public` | `private`）
- `template`: 模板 id

### 「我」控制台（精簡）

- 三角色選擇（成長／工廠／保護感）
- **狀態**：點一下切公開／私人（**展示用**，不再跳出模板對話框）
- **存檔**：顯示最後存檔（含模板名·v）
- **拼圖模板**：三個下載鈕（8×8／10×10／12×12），不開套用對話框
- 保留：排版紀錄、打包／還原
- **已從選單拿掉**：存拼圖位置、重置、鑰匙、AI 鑰匙、模板完整設定（header 異動時仍可存／重置）

### 種子內＝控制台編輯

- 點種子直接進**編輯模式**（不再先「讀」）
- 每一段是一個**框**；點框 → 底部對話框（鍵盤上方）輸入，不直接在畫面改
- 種子內 **doc-bar 暫時隱藏**（回拼圖／讀／改／存版等列先拿掉；可從 SEED 路徑與 🔔 快捷）

小精靈已併入鈴鐺；獨立選單已移除。

## 排版版本（JSON）

每次存檔會寫入兩處：

| 檔案 | 內容 |
|---|---|
| `docs/seeds.json` | 目前排版 + `map.savedAt`、`layoutName`、`layoutRev` |
| `docs/puzzle-layouts.json` | 依**模板**分組的歷史快照；每模板底下 v1、v2…（各模板最多 30 小版、最多 20 模板） |

**存檔行為**

- **Header 存檔圖示**：若已有模板名稱 → 直接存小版本（不跳對話框）
- **我 → 存拼圖位置**：一定開對話框；**留空** = 更新目前模板小版本；**輸入新名稱** = 另存新模板（與原本分開）
- 第一次存檔、留空 → 用預設時間名稱建立模板 v1

畫面：「我」裡排版紀錄顯示 `名稱 · v3`；對話框依模板分組列出小版本，可「用這版」還原。

## 版面最小化

- 拼圖格以**左右寬度**為主、固定 1:1 比例；拉寬視窗等同放大格子，高度超出時**上下滾動**
- 類型／存檔時間收在「我」popover
- 訊息用 toast + 🔔 通知
- 右下角 **小精靈**：常見按鈕 + 輸入框（AI 問答之後接上）

## 操作速記

- **右上**：選角色、看狀態／存檔、下載模板
- **SEED**：路徑 + 最近五處；右方顯示現在位置
- **點種子**：控制台框編輯（點段落 → 底部輸入）
- **永久保存拼圖位置**：畫面有異動時 header 存檔圖示（需 GitHub 鑰匙）
- **永久保存筆記**：🔔 → 存成一版（需鑰匙）
