# 知識拼圖（產品方向）

## 摘要

首頁不是「地理地圖」，而是**知識拼圖**：種子像拼圖片，擺在格子上組成畫面。東南西北拿掉。模板可自由選。

## 與舊說法的關係

| 舊 | 新 |
|---|---|
| 知識地圖 | **知識拼圖** |
| 東南西北方位 | **不使用** |
| 固定一種格子 | **可選拼圖模板** |

台灣橫貫公路示範仍可當「一種模板」，不再暗示真實方位。

## 兩種拼圖

### 1. 個人版

- 自己的拼圖工作台
- 可設定 **公開** 或 **私人**
- 位置自己拖、自己存
- 對應現況 Pages／seeds.json 流程

### 2. 社群版（像菜市場攤位）

- 大家搶同一張拼圖上的格子
- **位置用點數去佔**（熱門位置較貴或需較多點）
- 點數來源（之後實作）：
  - 完成任務（煉筆記、存版、分享、協助改稿…）
  - 隨時間緩慢累積
- 搶到的位置可展示自己的 seed；被更強點數擠走的規則之後再定

## 拼圖模板

使用者可自由選模板（第一版已有選單入口）：

| 模板 | 說明 |
|---|---|
| 空白 8×8／10×10／12×12 | 純格子，自己擺 |
| 示範：台灣橫貫公路 | 現有示範種子擺法 |

之後可加：時間軸、主題島、樹狀、圓環等模板，不必綁地理。

## 資料欄位（seeds.json 的 map）

```json
{
  "title": "知識拼圖",
  "kind": "personal",
  "visibility": "public",
  "template": "grid-10",
  "cols": 10,
  "rows": 10
}
```

- `kind`: `personal` | `community`
- `visibility`: 僅個人版有意義（`public` | `private`）
- `template`: 模板 id

## 排版版本（JSON）

每次「存拼圖位置」會寫入兩處：

| 檔案 | 內容 |
|---|---|
| `docs/seeds.json` | 目前排版 + `map.savedAt`（最後存檔時間） |
| `docs/puzzle-layouts.json` | 歷史快照（最多 30 版）；可「用這版排版」還原 |

畫面：header 左側 chip 顯示類型與同步狀態（綠＝已存、橙＝有異動）；有異動時 header 出現存檔／重置圖示。詳情點 chip 或選單「排版紀錄」。

## 版面最小化

- 拼圖格佔滿主畫面；類型／存檔時間收在 header chip 下拉
- 訊息用 toast + 🔔 通知
- 右下角 **小精靈**：常見按鈕 + 輸入框（AI 問答之後接上）

## 操作速記

- **選單**：拼圖模板、存拼圖位置、重置、鑰匙、打包／還原
- **點種子**：才出「回拼圖、讀、改、存版、請 AI 改…」
- **永久保存模板／位置**：選單 → 存拼圖位置（需 GitHub 鑰匙）
