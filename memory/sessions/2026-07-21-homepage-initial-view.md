# 開啟首頁初始視圖

## 摘要

開啟網站一律先落在首頁棋盤。顯示模式優先序：本機偏好 → 倉庫 `map.view` → 全部棋盤。目前倉庫存的是「上下為主」。

## 重點

- 啟動時明確 `showPanel("list")`，不再卡在其他面板。
- `loadCatalog` 會讀取 `seeds.json` 的 `map.view` 當共用初始視圖。
- HTML 首屏 chips／`map-shell` 對齊倉庫目前的 `height`，減少載入閃爍。
- 使用者切換後仍寫入本機；存棋盤時一併寫回倉庫。

## 決定／偏好

- 首頁＝進入點；初始視圖以倉庫為準，本機可覆蓋。

## 待續

- 手機實機確認首屏棋盤與三種顯示模式切換。
