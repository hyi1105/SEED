# 角色選擇＋控制台體感

## 摘要

右上「我」改成三角色選擇（像對戰選角）；SEED 首頁也可開路徑並顯示現在位置與最近五處；進種子直接框編輯（點段落→底部輸入），種子內操作列先隱藏。

## 重點

- 三角色：守護／工頭／偵察 — 保護自信、成長工廠感
- 狀態點一下切私人／公開（展示）；模板改三個下載；拿掉選單裡存檔／重置／鑰匙等
- SEED 右側顯示現在位置；下拉＝路徑階梯＋最近五處
- 每一段一個框，不直接在畫面編輯

## 決定／偏好

- 先試這套體感；進階設定（鑰匙等）仍藏在 DOM，之後再接入口
- doc-bar 功能暫時拿掉，改靠路徑與 🔔

## 待續

- 存成一版／鑰匙入口要更明顯（現靠 🔔）
