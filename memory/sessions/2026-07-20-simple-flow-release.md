# 簡化成 System → User → AI

## 摘要

使用者要求把目前互動收斂成最小流程：不要 system feedback board、不要 player records、不要一般 packaging instructions。只保留 **System → User → AI**。

## 重點

- 隱藏系統回饋板入口
- 隱藏排版紀錄入口
- 隱藏打包／還原入口
- AI 對話框改成兩欄：`System 先說`、`User 回答`
- AI 直接根據這兩段產生回答，不再預設改整份稿

## 決定／偏好

- 問答流程比功能面板更重要
- 包裝與紀錄可留在底層，但不應成為主畫面主流程

## 待續

- 若之後要把 AI 回答寫回筆記，再決定「套用回答」的方式
