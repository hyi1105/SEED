# 手機版 Seed 介面簡化

## 摘要

依手機使用情境簡化首頁與建立 Seed 流程：入口改為 **Seed**／**建立 Seed**，點進去只剩三個類型按鈕並直達編輯；隱藏棋盤顯示模式與重複的 doc-bar。文件範本改為**欄位設計器**（非填寫），每個欄位可獨立設定可見對象。

## 重點

- 移除獨立 **Seed** 區塊；**建立 Seed** 已拿掉，僅保留「＋ 新增 SEED」
- 掌托底部加**支撐栓**與捲動留白，整頁上拉時「我的 SEED」可捲到棋盤上方
- Seed 面板：僅 **文件／簽核／討論** 三鍵，點即開空白範本編輯
- 隱藏首頁棋盤顯示模式（左右／上下／全部棋盤三鍵）；棋盤本身保留
- 保留模式列：**編輯／文件檢視／歷程記錄**
- 移除 **另存為 System Seed**（編輯列、版本歷程）
- 首頁 palm rest 區（棋盤＋Seed＋我的 SEED）可上下捲動
- **文件（人員卡片 v2）**：
  - 預設欄位：簽名1、簽名2、簽名3（可改名、可新增）
  - 每欄獨立可見性：本人／所有人／＋指定人員（整行按鈕）
- **v2（同日晚）**：三列版面——類型（單行／多行／下拉）→ 欄位名 → 可見＋可編輯權限；下拉支援手動或 CSV／Excel；Save 後回首頁建版
- **v3**：每欄改為 status 列版型——第 1 列灰字「名稱／類別」、第 2 列藍字左名稱右類別、第 3 列左可編輯右可見
  - 拿掉：背景卡、照片、填寫值、全域權限、歷程面板、「文件格式」標籤

## 決定／偏好

- 連接 palm rest 先保留不動
- 歷程記錄模式列暫不搬動
- 這是**原版文件範本設計**，不是實例填寫
