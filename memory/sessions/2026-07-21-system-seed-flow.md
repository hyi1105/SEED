# System Seed 三層建立流程

## 摘要

新增 SEED 不再直接建立空白檔，改從首頁 **System Seed** 依類型、範本、版本逐步進入，修改後 Save 才成為「我的 SEED」。

## 流程

1. 首頁 → **System Seed**（或「＋ 新增 SEED」）
2. 選類型：文件／簽核／討論
3. 選範本（內建或自訂）
4. 選版本 → **用這版建立 SEED**
5. 進入編輯 → 修改 → **Save**

## 另存為範本

- 編輯中：**另存為 System Seed**
- 歷程記錄：任一版本可 **存成 System Seed**
- 範本列表頁也可把目前 SEED 另存為同類型範本

## 路徑

`首頁 › System Seed › 文件 › 空白文件`（依層級顯示）

## 待續

- 範本與版本目前存瀏覽器；正式多使用者需後端
- 可從棋盤既有 SEED「跳出」為範本的一鍵入口
