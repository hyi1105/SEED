# 書面文件改成分類＋合約書模板設計

## 摘要

「書面文件 Seed」改為**分類**。分類內有「合約書」，連到既有 form-template 填寫流程。分類擁有者可編輯公版模板，決定放文字／填空／下拉／簽名。

## 重點

- ＋新增 → 書面文件 Seed → 分類列表（不再直接開一份）
- 列表項目：**合約書** → 依公版開填寫實例
- 擁有者：**編輯合約書模板** → 文件預覽＋下方元件托盤（拖移／點選插入文字、填空、下拉、簽名），可拖移重排
- 公版存 `localStorage` key `written-form-design-v1`；新開合約書讀這份
- 填寫模式新增 `select` 下拉；固定文字仍不可改

## 決定／偏好

- 仍用 `document` + `documentLayout: "form-template"`，不新增頂層 seedType
- 個人 MVP 本機視為分類擁有者（之後可接真正 owner 權限）
- 模板編輯互動比照「加表情符號」：托盤拖進文件／點選插入

## 待續

- 多份書面子類型（不只合約書）
- 公版真正跨裝置／成員共享
- 手機觸控拖移再加強（目前 HTML5 drag + 點選插入）
