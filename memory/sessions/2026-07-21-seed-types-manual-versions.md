# 三種 SEED＋手動版本＋棋盤

## 摘要

SEED 從單一文件編輯器擴充為文件、簽核、主題討論三類；正式版本改為按 Save 才建立。首頁的「拼圖」改稱「棋盤」，我的 SEED 可拖進棋盤，也可從棋盤拖回。

## 重點

- 記錄比對只選一個編輯日期，自動與上一次版本比較。
- 版本顯示作者、由人或 AI 編輯、日期與時間。
- 編輯中的內容只暫存草稿；按 Save 才寫成正式版本。
- 文件模板使用較緊湊的新增文字介面。
- 簽核模板可設定文字欄位、下拉選單與預設簽核人。
- 主題討論模板使用群組對話，AI 可整理 A4 初版，匯出保留對話紀錄。
- A4 檢視提供 PDF、Word、圖片匯出。
- 回首頁後，頂部階層只顯示「首頁」。
- 我的 SEED 與棋盤支援雙向拖放。

## 決定／偏好

- 棋盤位置可自動儲存；文件版本不可自動建立。
- AI 產生的討論初版若直接 Save，版本標記為 AI；人再修改則標記為人。
- 第一版新建 SEED 先存在本機；既有倉庫文件維持文件模板。

## 驗證

- 本機瀏覽器建立簽核模板，設定簽核人並連續 Save 兩次。
- 確認記錄比對自動比較相鄰版本，差異為新增一行、刪除一行。
- 確認 A4 顯示三個匯出按鈕。
- 確認首頁階層只有「首頁」、棋盤 100 格與 8 顆既有 SEED 正常載入。
- 測試 SEED 已刪除；本次未發布。
