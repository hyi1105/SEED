# 兩欄編輯＋文件匯入

## 摘要

文件編輯改成左側工具、右側正文；首頁增加三種棋盤顯示比例。文件檢視可匯入 Word、PDF 與文字資料，歷程記錄則預設比較目前草稿和上次 Save。

## 重點

- Save 只在編輯畫面出現，位於「編輯」左側。
- 標題與副標題可直接修改。
- 左欄顯示段落編號、註解、新增、刪除與拖曳把手；右欄只放正文。
- 首頁直接切換「左右為主／上下為主／全部棋盤」，預設全部棋盤。
- 棋盤和我的 SEED 之間只用拖曳，不再提供收起或派遣圖示。
- A4 檢視改名文件檢視，新增檔案匯入。
- 歷程記錄沒有確認按鈕，進入後立即比較目前內容與上次 Save。

## 文件往返原則

- Word `.docx` 透過 Mammoth 抽取文字；PDF 透過 PDF.js 抽取文字。
- TXT、Markdown、CSV、TSV、JSON 直接解碼為文字。
- 原始 Word／PDF 位元保存在 IndexedDB。
- 未修改時，匯出同格式會下載原檔，可完全一致。
- 內容、標題或副標題修改後，改由目前 SEED 重新產生，無法保證原排版完全相同。

## 驗證

- 確認首頁預設為全部棋盤，另兩種模式可直接切換。
- 確認棋盤收起圖示與我的 SEED 派遣按鈕皆為零。
- 確認 Save 僅在編輯出現，文件檢視和歷程記錄不顯示。
- 確認兩欄編輯器、標題與副標題欄位正常。
- 確認歷程預設值為目前畫面，且自動完成比對。
- 確認 Mammoth 與 PDF.js 可載入，CSV 文字抽取正常。

## 09:25 體感調整

- 頂部只保留三個模式；Save 改到編輯內容的第二行，與文件匯入／匯出同為非固定小工具列。
- 文件模板移除說明性標題，改以灰色小字顯示 `版型：文字模板`。
- 左側改成服務右側正文的獨立註解對話框；段落編號使用綠色圓圈，不再顯示移動圖示。
- 正文框靠近排列；刪除放右上角，新增以淡線中央的灰底白字圓形 `＋` 呈現。
- 歷程記錄標題與下拉選單限制為 48rem，和下方 A4 同寬。
- 「上下為主」及「全部棋盤」皆隱藏垂直捲軸。
- 瀏覽器確認工具列 `position: static`、歷程控制與紙張均為 864px，且兩個棋盤模式的垂直 overflow 均為 hidden。

## 09:47 直播截圖＋操作空間

- 「我的 SEED」保留至少 7rem 操作高度，清單過長時在區域內上下捲動。
- 新增 SEED 改成直接選文件／簽核／討論，選完以預設名稱建立並立即進入編輯。
- 討論模板不再是聊天紀錄或 AI A4 初稿，改為 16:9 直播截圖製作器：
  - 標題、副標題、主要文字都呈現在圖內。
  - 左下疊加最新四則評論。
  - 圖片匯出使用 1280×720 畫布。
- Save 改成鮮明綠色；文件匯入與匯出使用不同的柔和色組。
- 文件段落左側註解改成與正文相同框形，編號為純文字；新增 `＋` 移入正文右上刪除 `×` 下方。
- 歷程記錄移除頂部大標題，版本文字改成 `編輯者－編輯－日期時間`。
- 本機瀏覽器已驗證快速建立、直播預覽、左下評論、段落控制位置、歷程格式與 SEED 清單捲動設定。

## 10:46 簽核工作流原型

- 編輯模式的正式版本 `Save` 移到整份內容最下方。
- 簽核模板擴充為五個頁籤：申請單、欄位、流程、權限、通知。
- 下拉選項可使用手動值或 CSV／TSV／Excel；支援指定 A／B／C 欄、標題列與允許手動輸入。
- 加入 Requester、Filler、Approver、CopyTo、FYI、Admin、Owner 角色。
- 加入依序／平行簽核、全數／任一通過、Return、Delegate、期間代理、Reminder 及超時動作設定。
- Owner 可使用欄位 × 角色表格大量設定可看、可編輯、必填與條件／lookup。
- 申請單以對話式欄位呈現，支援草稿、Submit、Approve、Deny、Return、Delegate、Call 與 Comment。
- 流程圖可點人頭切換角色，箭頭可進入通知設定；通知區分固定前後段與可編輯中段。
- 前端已保存 Status、Current Approver、Current Level、Last Submit Date、Last Approval Date 及操作紀錄。
- 真實身分授權、寄信、排程、代理、多人交易與不可竄改稽核仍需後端。
- 瀏覽器確認五個頁籤、角色流程、權限矩陣、通知模板、CSV 指定欄解析與 Submit 狀態轉換。
