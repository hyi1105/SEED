# System Seed 文件直接編輯

## 摘要

文件類型與簽核相同：System Seed → 文件 → 空白文件，點進去直接開編輯器，不再經版本層。

## 畫面內容

同一畫面可見：

- **文件格式**：文字模板
- **文件說明**：可編輯（存於 `blurb`）
- **段落編輯**：沿用既有框編輯
- **歷程**：最近版本列表，可預覽、用這版繼續改，或開完整歷程記錄

## 決定／偏好

- Save 後歷程區自動更新
- 快取 `control14`
