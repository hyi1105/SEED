      選擇棋盤模板

      模板決定格子大小與擺法。之後可加更多樣式；現在先這幾種。

      個人版可設公開或私人。套用模板會改格子大小；記得再「存棋盤位置」。

      搶這一格

      這格要幾點

      設定 GitHub 鑰匙

      要把地圖位置或筆記內容寫進倉庫，需要權杖。只存在這台電腦的瀏覽器，不會進網頁原始碼。

      1. 打開 Fine-grained tokens
      2. Generate → 只選 repo SEED → Permissions → Contents：Read and write
      3. 產生後貼到下面（不要貼到聊天）

      會員碼

      付費後會拿到一組碼。阿嬤只要貼上這組碼，不用申請 AI key。

      設定 AI 鑰匙

      「AI 回答」會呼叫 OpenAI（或相容服務）。鑰匙只存在這台瀏覽器，不要貼到聊天。

      1. 到 OpenAI API keys 建立一把 key
      2. 貼到下面（通常以 sk- 開頭）

      System → User → AI

      先放 system 的問題或情境，再寫 user 的回答；AI 只根據這兩段回應，不另外顯示回饋板或玩家紀錄。

      排版紀錄
