設定 GitHub 鑰匙

要把地圖位置或筆記內容寫進倉庫，需要權杖。只存在這台電腦的瀏覽器，不會進網頁原始碼。

  1. 打開 Fine-grained tokens
  2. Generate → 只選 repo SEED → Permissions → Contents：Read and write
  3. 產生後貼到下面（不要貼到聊天）

存成一版

幫這一版取個好記的名字（例如「初稿」「補了坑」）。會寫進倉庫，之後可在「回到舊的」找到。
